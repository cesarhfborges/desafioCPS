## Como Iniciar
Antes de iniciar a manipulação de codigo é necessário baixar as dependencias do projeto.
```
npm install
```

## Como Rodar
O projeto automaticamente abre o navegador ao finalizar o start
```
npm start
```

## Build

Build de producao do sistema  
```
ng build --prod --aot
```

Build do sistema
```
ng build  --aot
```
